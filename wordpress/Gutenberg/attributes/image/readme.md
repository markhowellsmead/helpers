# How to use an image object as an attribute

## In JavaScript

```javascript
attributes: {
    image_default: {
        type: 'Object',
        default: {
            id: false,
        },
    },
},
 ```

## In PHP


```php
'image_default' => [
    'type' => 'object',
    'default' => [
      'id' => false,
    ]
],
```
